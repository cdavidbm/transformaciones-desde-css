Unidades Absolutas en CSS

Las unidades absolutas tienen un tamaño fijo independientemente del dispositivo o contexto. Son útiles para elementos que necesitan dimensiones específicas.

Píxeles (px)

200px × 100px

Unidad más común para pantallas

Puntos (pt)

150pt × 75pt

Usado principalmente en impresión

Pulgadas (in)

2in × 1in

1 pulgada = 96px

Centímetros (cm)

5cm × 3cm

1cm = 37.8px

Milímetros (mm)

50mm × 30mm

1mm = 3.78px
