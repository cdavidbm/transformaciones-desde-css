Unidades Flexibles en CSS

Unidad fr (Fraction)

¿Qué es fr?

La unidad fr representa una fracción del espacio disponible en un contenedor grid. Es perfecta para crear layouts flexibles y responsivos.

Unidad ch (Character)

¿Qué es ch?

La unidad ch representa el ancho del carácter "0" en la fuente actual. Es útil para crear campos de texto con anchos específicos en caracteres.

Este texto está limitado a 40 caracteres de ancho. Lorem ipsum dolor sit amet, consectetur adipiscing elit.

Uso práctico: Ideal para campos de formulario, código, y texto monoespaciado.

Unidad ex (X-Height)

¿Qué es ex?

La unidad ex representa la altura de la letra "x" minúscula en la fuente actual. Se usa principalmente para ajustes tipográficos finos.

Texto normal (24px) 10ex × 5ex más texto
Texto más grande (36px) 10ex × 5ex más texto

Nota: Observa cómo la caja se escala proporcionalmente con el tamaño de fuente.

Comparación de Unidades Flexibles

Cuándo usar cada unidad:

  • fr: Layouts de grid flexibles, distribución proporcional de espacio
  • ch: Campos de texto, contenido monoespaciado, límites de caracteres
  • ex: Ajustes tipográficos finos, elementos relacionados con la altura de letra

Tip: Las unidades flexibles se adaptan automáticamente a cambios en el contexto, lo que las hace perfectas para diseños responsivos y accesibles.
