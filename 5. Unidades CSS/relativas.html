Unidades Relativas en CSS

Las unidades relativas se adaptan según el contexto (tamaño de fuente, elemento padre, viewport). Son ideales para diseños responsivos.

Contenedor padre (font-size: 20px)

Unidad em

10em × 5em

Relativo al font-size del elemento padre

Unidad rem

15rem × 8rem

Relativo al font-size del elemento raíz (html)

Contenedor padre (300px × 200px)

Porcentaje (%)

80% × 60%

Relativo al elemento padre

Unidades de Viewport

50vw × 30vh

vw = viewport width, vh = viewport height

Consejos:

  • em: Hereda del elemento padre, útil para componentes escalables
  • rem: Consistente en toda la página, mejor para tipografía
  • %: Perfecto para layouts fluidos y responsivos
  • vh/vw: Ideales para elementos que ocupan porcentajes de la pantalla
