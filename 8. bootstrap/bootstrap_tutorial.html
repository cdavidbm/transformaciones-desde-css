Guía Visual de Bootstrap 5

Aprende a usar Bootstrap con ejemplos prácticos, ¡creado 100% con Bootstrap!

1. Sistema de Rejilla (Grid System)

El sistema de rejilla es la base para crear layouts responsivos. Usa una combinación de contenedores, filas (`row`) y columnas (`col`).

Una columna (col-sm)
Una columna (col-sm)
Una columna (col-sm)
Ocupa 8 de 12 columnas en pantallas medianas (`md`) o más grandes
Ocupa 4 de 12

Código de ejemplo:

<!-- Tres columnas iguales en pantallas pequeñas o más grandes -->
<div class="row">
  <div class="col-sm">Columna 1</div>
  <div class="col-sm">Columna 2</div>
  <div class="col-sm">Columna 3</div>
</div>

<!-- Columnas con anchos específicos -->
<div class="row mt-3">
  <div class="col-md-8">Columna de 8</div>
  <div class="col-md-4">Columna de 4</div>
</div>

2. Botones

Estilos de botones predefinidos para diferentes acciones. Simplemente aplica la clase `btn` y una clase de estilo como `btn-primary`.

Código de ejemplo:

<button type="button" class="btn btn-primary">Primario</button>
<button type="button" class="btn btn-success">Éxito</button>
<button type="button" class="btn btn-danger">Peligro</button>

3. Tarjetas (Cards)

Un contenedor de contenido flexible y extensible. Ideal para mostrar información en bloques.

Título de la Tarjeta

Este es un texto de ejemplo dentro del cuerpo de la tarjeta para demostrar su funcionalidad.

Ir a algún lado

Código de ejemplo:

<div class="card" style="width: 18rem;">
  <img src="..." class="card-img-top" alt="...">
  <div class="card-body">
    <h5 class="card-title">Título de la Tarjeta</h5>
    <p class="card-text">Texto de ejemplo para la tarjeta.</p>
    <a href="#" class="btn btn-primary">Ir a algún lado</a>
  </div>
</div>

4. Alertas

Proporciona mensajes de retroalimentación contextuales para acciones típicas del usuario.

¡Esto es una alerta de éxito!
¡Esto es una alerta de peligro!
¡Esto es una alerta de advertencia!

Código de ejemplo:

<div class="alert alert-success" role="alert">
  ¡Esto es una alerta de <strong>éxito</strong>!
</div>
<div class="alert alert-danger" role="alert">
  ¡Esto es una alerta de <strong>peligro</strong>!
</div>

5. Barra de Navegación (Navbar)

Un componente de navegación responsivo que se colapsa en pantallas pequeñas.

Mi Sitio
  • Inicio
  • Características
  • Deshabilitado

Código de ejemplo:

<nav class="navbar navbar-expand-lg bg-body-tertiary">
  <div class="container-fluid">
    <a class="navbar-brand" href="#">Mi Sitio</a>
    <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarNav">
      <span class="navbar-toggler-icon"></span>
    </button>
    <div class="collapse navbar-collapse" id="navbarNav">
      <ul class="navbar-nav">
        <li class="nav-item">
          <a class="nav-link active" href="#">Inicio</a>
        </li>
        <li class="nav-item">
          <a class="nav-link" href="#">Características</a>
        </li>
      </ul>
    </div>
  </div>
</nav>

6. Formularios

Estilos para controles de formulario, etiquetas y mensajes de validación.

Nunca compartiremos tu correo con nadie más.

Código de ejemplo:

<form>
  <div class="mb-3">
    <label for="exampleInputEmail1" class="form-label">Correo Electrónico</label>
    <input type="email" class="form-control" id="exampleInputEmail1">
  </div>
  <div class="mb-3">
    <label for="exampleInputPassword1" class="form-label">Contraseña</label>
    <input type="password" class="form-control" id="exampleInputPassword1">
  </div>
  <button type="submit" class="btn btn-primary">Enviar</button>
</form>

7. Modal

Una ventana de diálogo que se superpone al contenido de la página. Requiere el JavaScript de Bootstrap.

Título del Modal

Este es el contenido del cuerpo del modal. ¡Todo esto se muestra sin recargar la página!

Código de ejemplo:

<!-- 1. Botón que activa el modal -->
<button type="button" class="btn btn-success" data-bs-toggle="modal" data-bs-target="#exampleModal">
  Lanzar Modal
</button>

<!-- 2. Estructura del Modal -->
<div class="modal fade" id="exampleModal" tabindex="-1">
  <div class="modal-dialog">
    <div class="modal-content">
      <div class="modal-header">
        <h5 class="modal-title">Título del Modal</h5>
        <button type="button" class="btn-close" data-bs-dismiss="modal"></button>
      </div>
      <div class="modal-body">
        <p>Contenido del cuerpo del modal.</p>
      </div>
      <div class="modal-footer">
        <button type="button" class="btn btn-secondary" data-bs-dismiss="modal">Cerrar</button>
      </div>
    </div>
  </div>
</div>
